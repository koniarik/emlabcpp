Detailed Description

result represents an result of some operation, as an alternative to returning just bool with true/false value.

Result has only two states, SUCCESS or ERROR, to check which state it is in use operator== or operator !=: if( res == SUCCESS) { ..., alternativelly, one can directly access the state value via get_state(), which returns numerical representation.
