Detailed Description

template<typename Container>
class emlabcpp::iterators::subscript_iterator< Container >

Subscript iterator stores reference to given container and index of item.

The item is referenced via the operator[] of said container. Keep that in mind, as it can behave counter-inuitivelly in case you modify the container while you iterate it
