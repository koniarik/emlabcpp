29 // Register map is abstraction to work with registers of external devices. It stores values of
30 // serializable types that can be accessed based on key (usually enum representing address of
31 // register in the device).You can access the value based on the key itself, both at compile time
32 // and at runtime. You can also iterate over the values and there is handler that allows
33 // serialization and deserialization of bytes into the values defined in the map. This includes
