Detailed Description

template<typename Callable>
class emlabcpp::defer< Callable >

Defer is an object that receives and stores callable, which is called on it's destruction.

The mechanism should be used to setup callbacks executed on end of scope.

For example:

void foo(){ defer d = []{ std::cout << "tree" << std::endl; }; }

Prints "tree" once foo() finishes it's execution
